# DeV4GiRLS_2026
Proyecto educativo a largo plazo para que una niña de 12 años aprenda a pensar como ingeniera creando y publicando una app móvil sencilla con minijuegos. Los juegos son el medio para enseñar arquitectura, sistemas, criterio técnico y uso crítico de IA, priorizando comprensión, diseño y juicio sobre “que funcione”.
